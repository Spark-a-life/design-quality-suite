# Surface Reference -- Landing and Editorial

Use this reference for marketing pages, public launches, editorial stories, and
brand-forward surfaces.

## Priorities

1. Strong first impression
2. Narrative flow
3. Clear calls to action
4. Controlled distinctiveness

## Layout guidance

- Use asymmetry and whitespace intentionally.
- Avoid default three-column feature grids when a stronger rhythm is available.
- Keep the hero structurally clear rather than stacking everything in the center.

## Visual guidance

- One accent family is still the default.
- Typography should carry part of the visual identity.
- Texture, gradients, and imagery should support the message rather than mask a
  weak layout.

## Interaction guidance

- Motion may be more expressive than in dashboards, but still must remain
  performant and comprehensible.
- CTAs should feel deliberate and tactile, not generic.
